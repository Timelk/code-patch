Refactor Component

This skill analyzes a selected React component and suggests improvements for performance, readability, and modern React patterns (e.g., hooks, functional components).

Prompt Template

ACT AS a Senior React Developer.
TASK: Refactor the following code snippet.
CONSTRAINTS:
- Use functional components.
- Implement {{state_management}} for state.
- Ensure TypeScript types are explicit.
CODE:
{{selected_code}}

Usage

Select a block of code in your editor, right-click, and choose "Code Patch: Refactor Component". The output will be displayed in a diff view.

Parameters

  • state_management: (Optional) Specify 'Redux', 'Context', or 'Zustand'. Defaults to 'useState'.
  • styling: (Optional) Specify 'Tailwind', 'CSS Modules', or 'Styled Components'.
